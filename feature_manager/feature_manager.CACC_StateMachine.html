feature_manager.CACC_StateMachine module

State machine for cooperative adaptive cruise control transitions.

This module defines the CACC feature manager state machine and maps cruise control request messages to driver event inputs.

class StateMachineForCACC

Bases: object

Manages transitions for CACC across OFF, STBY, ON, and FAULTED states.

property state

Current state of the CACC feature.

Returns:

The current state of the underlying state machine.

Return type:

State

step()

Advance the state machine by one evaluation cycle.

The FSM evaluates all enabled transitions once and then clears one-shot command events so that they do not persist into the next step.

reset_events()

Clear one-shot driver event flags.

This ensures a cruise request event is only acted upon during the step in which it was received.

update_fields_with_cruise_request(msg)

Translate a CruiseRequest message into internal event flags.

Parameters:

msg (CruiseRequest) – Incoming cruise control request message.
